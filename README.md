# Habit Tracker

A beautiful and modern habit tracking application built with React and TypeScript.

## Features

- ✅ Add and manage multiple habits
- ✅ Track habits daily with a simple toggle
- ✅ Two view modes: List view and Calendar view
- ✅ Navigate through dates easily
- ✅ Color-coded habits for visual organization
- ✅ Persistent storage using localStorage
- ✅ Beautiful, modern UI with gradient background
- ✅ Fully responsive design

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`)

### Building for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

To preview the production build:

```bash
npm run preview
```

## Usage

1. **Add a Habit**: Type a habit name in the input field and click "Add Habit"
2. **Mark Complete**: Click the circle button next to a habit to mark it as completed for the selected date
3. **Change Date**: Use the arrow buttons or click "Today" to navigate to different dates
4. **Delete Habit**: Click the "×" button to remove a habit
5. **Switch Views**: Use the "List" and "Calendar" buttons in the header to toggle between views
6. **Calendar View**: See all your habits across the month with color-coded dots indicating completion

## Technologies Used

- React 18
